### 说明

二次元语录给你力量

- 插件类型: `NORMAL`
- 响应规则: `全局`
- 插件等级: `5`

### 指令

- 语录 | 二次元
  - 说明: 随机发送二次元语录

### 示例

<Messenger
  msgs={[
    { position: "right", msg: "语录" },
    {
      position: "left",
      msg: `音无结弦之日，跃动天使之心，立于浮华之世，奏响天籁之音。	——Angel Beats!`,
    },
    { position: "right", msg: "二次元" },
    {
      position: "left",
      msg: `窃人者，自失其宝。	——假面骑士kabuto`,
    },
  ]}
/>
