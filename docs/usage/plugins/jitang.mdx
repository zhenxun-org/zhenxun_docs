import Messenger from "/src/components/Messenger"

### 说明

大饼吃多了，那就来喝点鸡汤吧~

- 插件类型: `NORMAL`
- 响应规则: `全局`
- 插件等级: `5`

### 指令

- 鸡汤
  - 说明: 随机发送心灵鸡汤

### 示例

<Messenger
  msgs={[
    { position: "right", msg: "鸡汤" },
    {
      position: "left",
      msg: `你正在成为自己需要的那个人`,
      reply: "鸡汤",
    },
    { position: "right", msg: "鸡汤" },
    {
      position: "left",
      msg: `温柔是你内在的力量。`,
      reply: "鸡汤",
    },
    { position: "right", msg: "鸡汤" },
    {
      position: "left",
      msg: `有时候暂停也是前进的一部分。`,
      reply: "鸡汤",
    },
  ]}
/>
