### 说明

为你点播了一首简单的歌

- 插件类型: `NORMAL`
- 响应规则: `全局`
- 插件等级: `5`

### 指令

- 在线点歌
  - 说明:  点歌 [歌名]

  msgs={[
    { position: "right", msg: "点歌 ひめごと*クライシスターズ" },
    {
      position: "left",
      imgSrc: "docs/builtin_plugins/music/0.jpg"
    },

  ]}
/>
