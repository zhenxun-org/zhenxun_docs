### 说明

这是一个功能强大的B站内容解析插件，支持视频、直播、专栏/动态、番剧等多种B站内容的解析和下载功能。

- 插件类型: `DEPENDANT`
- 响应规则: `全局`
- 插件等级: `5`

### 指令

- [B站内容解析插件使用指南](https://github.com/zhenxun-org/zhenxun_bot_plugins/blob/main/plugins/parse_bilibili/README.md)

### 示例

<Messenger
  msgs={[
    { position: "right", msg: "https://www.bilibili.com/video/BV1Ed4y1s7Ht/" },
    {
      position: "left",
      imgSrc: "docs/builtin_plugins/parse_bilibili/0.jpg",
    },

  ]}
/>
