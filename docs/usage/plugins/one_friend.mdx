### 说明

我有个朋友想说...

- 插件类型: `NORMAL`
- 响应规则: `全局`
- 插件等级: `5`

### 指令

- 我有一个朋友想问问
  - 说明: 我有一个朋友想问问 [文本] ?[at]: 当at时你的朋友就是艾特对象

### 示例

<Messenger
  msgs={[
    { position: "right", msg: "我有一个朋友想问问 这个朋友是不是你自己?@_" },
    {
      position: "left",
      imgSrc: "docs/builtin_plugins/one_friend/0.jpg"
    },

  ]}
/>
