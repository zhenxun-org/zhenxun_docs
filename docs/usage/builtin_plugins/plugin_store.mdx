import Messenger from "/src/components/Messenger"

### 说明

通过指定仓库安装真寻插件

- 插件类型: `SUPERUSER`
- 响应规则: `全局`
- 插件等级: `5`

### 配置项

无

### 指令

- 插件商店

- 添加插件 [id/module]

  - 说明: 安装指定插件，id 和 module 通过插件商店获取

- 移除插件 [id/module]

  - 说明: 安装指定插件，id 和 module 通过插件商店获取

- 更新插件 [id/module]

  - 说明: 安装指定插件，id 和 module 通过插件商店获取

- 搜索插件 [s]

  - 说明: 通过插件名称或作者搜索插件

- 更新全部插件
