### 说明

个人昵称，将替换小真寻称呼你的名称，群聊 与 私聊 昵称相互独立，  
全局昵称设置将更改你目前所有群聊中及私聊的昵称

- 插件类型: `NORMAL`
- 响应规则: `全局`
- 插件等级: `5`

### 配置项

- `nickname`
  - `BLACK_WORD`:
    - 说明: 昵称所屏蔽的关键词，已设置的昵称会被替换为 \*，未设置的昵称会在设置时提示
    - 类型: `list[str]`
    - 默认值: `["爸", "爹", "爷", "父"]`

### 指令

- 以后叫我[昵称]
  - 说明: 设置当前群聊/私聊的昵称
  - 示例: 以后叫我椛椛
- 设置全局昵称[昵称]

  - 说明: 设置当前所有群聊和私聊的昵称

- 我是谁/我叫什么

  - 说明: 查询当前群聊/私聊的昵称

- 取消昵称
  - 说明: 取消当前群聊/私聊的昵称(真寻可能会伤心)

### 示例

<Messenger
  msgs={[
    { 
      position: "right", msg: "真寻以后叫我椛椛" },
    {

    },
  ]}
/>
