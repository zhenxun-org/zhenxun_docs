import Messenger from "/src/components/Messenger"

### 说明

检查更新真寻最新版本，包括了自动更新

- 插件类型: `SUPERUSER`
- 响应规则: `全局`
- 插件等级: `5`

### 配置项

无

### 指令

- 检查更新 ?['dev', 'main', 'resource'] ?[-r]
  - 说明: 在'dev', 'main'中更新其中一个版本, 可以使用-r 参数同时更新资源文件，当使用'resource'时单独更新资源文件

### 示例

<Messenger
  msgs={[
    { position: "right", msg: "检查更新 release" },
    {
      position: "left",
      msg: `版本更新完成
版本: v0.2.1 -> v0.2.2
请重新启动真寻以完成更新!`,
      reply: "检查更新 release"
    },
    { position: "right", msg: "检查更新 main -r" },
    {
      position: "left",
      msg: `版本更新完成
版本: v0.2.1 -> v0.2.2
请重新启动真寻以完成更新!
资源文件更新完成！`,
      reply: "检查更新 main -r"
    },
  ]}
/>
