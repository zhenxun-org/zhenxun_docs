import Messenger from "/src/components/Messenger"

### 说明

记录 好友/群组 请求，同时提醒超级用户请求信息

- 插件类型: `HIDDEN`
- 响应规则: `-`
- 插件等级: `-`

### 配置项

- `invite_manager`
  - `AUTO_ADD_FRIEND`:
    - 说明: 自动同意好友请求
    - 默认值: `False`

### 指令

- 无

### 示例

<Messenger
  msgs={[
    {
      position: "left",
      msg: `*****一份好友申请*****
ID: 1
昵称: 椛椛(123456789)
自动同意：×
日期: 2024-10-05 02:06:49
备注: 我是来自群聊“椛椛的小家”的昵称：椛椛`,
    },
    {
      position: "right",
      msg: `同意好友请求`,
      reply: `*****一份好友申请*****
ID: 1
昵称: 椛椛(123456789)
自动同意：×

备注: 我是来自群聊“椛椛的小家”的昵称：椛椛`
    },
    {
      position: "left",
      msg: `成功处理请求！`,
      reply: `同意好友请求`
    },
    {
      position: "left",
      msg: `*****一份入群申请*****
ID: 2
