### 说明

查看群组管理员帮助

- 插件类型: `ADMIN`
- 响应规则: `群组`
- 插件等级: `5`

### 配置项

- `admin_help`
  - `TYPE`
    - 说明: 管理员帮助样式，normal, zhenxun
    - 类型: `str`
    - 默认值: `zhenxun`

### 指令

- 管理员帮助
