### 说明

更新群组的基本信息

- 插件类型: `SUPER_AND_ADMIN`
- 响应规则: `群组`
- 插件等级: `1`

### 配置项

无

### 指令

- 更新群组信息

### 示例

<Messenger
  msgs={[
    { position: "right", msg: "更新群组信息" },
    {
      position: "left",
      msg: "已经成功更新了群组信息!",
    },
  ]}
/>
